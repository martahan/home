Reproducible Analyses for our CHEP 2019 paper Analysis Tools for the VyPR Performance Analysis Framework for Python

To make the results we present in papers reproducible, we place our analysis code (written using VyPR's analysis library) in Jupyter notebooks wrapped in Docker containers.

With this approach, it's very easy for our analyses to be rerun without investing a lot of time in setting up the necessary environment.

To reproduce the analysis discussed in our CHEP 2019 paper, you can find the Docker container files here.

This Docker container will run a Jupyter notebook, which will start up an instance of the VyPR server in the background to allow connection to a database that is destributed with the container. The port on which the Jupyter notebook serves is exposed, so you can access the notebook by connecting to localhost with whichever token the output from Jupyter gives you. From there, open the notebook plot_time_vs_measurement.ipynb.
