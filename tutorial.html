Setup

Here you can find instructions on how to run our tutorial, a self-contained way to introduce users, contributors and students to VyPR.

The source of the VyPR tutorial can be found on GitHub here.

The tutorial is tested on Linux (Scientific Linux, specifically CERN's distribution) and can be obtained by making a new, clean directory and, inside it, running

curl http://cern.ch/vypr/setup_tutorial.sh -L -o setup_tutorial.sh && chmod +x setup_tutorial.sh

and then executing the setup_tutorial.sh script.

Once the script finishes, you will have a new directory structure in the directory you set up, including VyPR's core infrastructure, its verdict server, and the analysis library.

The tutorial (a Jupyter notebook) can then be started by navigating to VyPRTutorial/ and running

jupyter notebook --ip 0.0.0.0 --port 9004 --allow-root

This then works like a standard Jupyter notebook - you can use it by navigating to http://localhost:9004.
