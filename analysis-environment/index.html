The VyPR Analysis Environment is a web-based tool that provides an intuitive interface for software engineers to visualise performance data collected by VyPR while monitoring software. While the Python library allows engineers to write scripts to automate their analyses, the web-based environment allows certain analyses to be performed with just a few clicks.
