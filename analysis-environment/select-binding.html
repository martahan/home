Upon the calls selection, the listing of the monitored function source code is compressed to show only the relevant bits of code. Apart from the lines detected by quantifiers, the calls selection enabled filtering the instrumentation points at which observations were taken during those calls and highlighting those points in the code as well. Due to the fact that a variable can be changed, or a function called, at multiple points in the code, the points of interest are grouped by bindings. Hence, the next step is to select a binding and see the points filtered to match the selection.
